\# Active Directory Infrastructure Lab



This project demonstrates the deployment of an enterprise-grade Active Directory environment including DNS, DHCP, Group Policy, and RADIUS services.



\## Technologies

\- Windows Server 2016

\- Active Directory DS

\- DNS / DHCP

\- Group Policy

\- Network Policy Server (RADIUS)



\## Use Case

Centralized identity and access management for corporate users and systems.
